## Concepts Learned

- **IPOS Architecture** — Input, Process, Output, Storage applied end-to-end
- **if / elif / else** — Full control flow branching with no silent logic gaps
- **Score Accumulator** — Integer state maintained across iterations (`score += 1`)
- **Input Sanitization** — `.strip()` + `.lower()` filter pipeline on all user input
- **f-strings** — Dynamic runtime interpolation with alignment formatting
- **Data Normalization** — Defending against casing inconsistencies (`"Paris" != "paris"`)
