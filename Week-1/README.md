## Concepts Learned

- **Lists** — dynamic arrays, `append()`, indexing, list comprehensions
- **Dictionaries** — key-value pairs, mutation, structured data (like DB rows)
- **File I/O** — reading and writing JSON for data persistence
- **Functions** — clean code separation using the IPO model
- **Control Flow** — `while` loops, `for` loops, `if/elif/else`
- **String Methods** — `.strip()`, `.lower()`, f-strings
